## Knowledge Architecture Principles

The folder hierarchy is expected to evolve.

The AI should never assume today's hierarchy is permanent.

When new knowledge is added:

1. Search existing documentation.
2. Find the most appropriate location.
3. If an existing page fits, merge into it.
4. If an existing category fits, place the page there.
5. If no category fits, recommend a new category rather than creating one automatically.
6. If moving pages would improve organization, propose the reorganization before making changes.
7. Never duplicate knowledge.
