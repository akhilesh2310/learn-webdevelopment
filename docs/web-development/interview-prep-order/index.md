# Interview Prep Order

## Fastest Interview Preparation Order

### JavaScript First

1. JavaScript Fundamentals
2. Execution Context
3. Closures
4. Objects & Prototypes
5. Async JS
6. Event Loop

### Then React

1. React Fundamentals
2. Component Lifecycle & Rendering
3. Hooks
4. State Management
5. Performance Optimization
6. React Internals
7. Architecture

### Then Next.js

1. Fundamentals
2. Routing
3. Rendering Strategies
4. Server vs Client Components
5. Data Fetching
6. Caching
7. Performance
8. Authentication

For Senior/Staff React interviews, the highest-yield sections are:

* React Fundamentals
* Hooks
* Rendering & Re-renders
* State Management
* Performance Optimization
* React Internals
* Architecture

Those 7 areas account for the majority of React interview discussions at your experience level.
