# Interview Q&A

## [https://namastedev.com/learn/namaste-interview](https://namastedev.com/learn/namaste-interview)

## JavaScript & TypeScript Interview Preparation Guide

This document is organized by topic for structured self-study. Each section includes commonly asked interview questions with clear, concise answers and code examples.

---

### 🧠 1. JavaScript Core Concepts

* What is the use of `isNaN()` function?

* What are undeclared & undefined variables?

* What is the difference between `null` & `undefined`?

* JavaScript type system: dynamic typing and type coercion (`==` vs `===`, implicit conversions).

* What is hoisting? Demonstrate with `var`, `let`, and `const`.

* What are IIFEs and why are they used?

* Use `call`, `apply`, and `bind` with examples.

* Implement a custom `bind()` method.

* Closures and lexical scope – explain with counter example.

* Currying vs Function Composition — Implement currying.

* Debounce vs Throttle – and implement both.

* Deep vs shallow copy – implement both (includes object cloning).

* Explain the `delete` operator.

* `var`, `let`, and `const` — scoping, redeclaration, and temporal dead zone.

* Static concept in JavaScript.

* Access modifiers in JS — what is "protected" and how to simulate it.
